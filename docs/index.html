⚡ workbench

Run multiple AI coding agents in parallel, each in its own git worktree, orchestrated from a terminal dashboard.

Recommended install

curl -fsSL https://fneves.github.io/workbench/install.sh | sh

Downloads the latest release, installs dependencies, and sets up your PATH. Works on macOS (arm64) and Linux (x64).

What it does

  • Spawns AI agents (Claude Code, OpenCode) in isolated git worktrees
  • Dashboard TUI shows all tasks, status transitions, diff stats
  • Per-task watcher with code review, diff viewer, git shortcuts
  • Notifications when agents finish, need input, or fail
  • Optional devcontainer mode for headless agents

Requirements

  • cmux — terminal multiplexer required
  • git required
  • jq required
  • claude or opencode — AI agent required
  • gh fzf lazygit delta bat optional

The installer auto-installs what it can via Homebrew (macOS) or apt (Linux).

Quick start

  • Open cmux
  • Run workbench start
  • Press n to spawn a new task

Verify install

workbench doctor
